1024 Jenkins重启Failed

发表于 2017-10-24 | 分类于 服务器

今天是1024，程序员的节日，也是十九大胜利闭幕的日子。
刚到公司，还没有开始干活，就听到了微信群里讲，Jenkins挂了。
自从方舟上线以来，jenkins就成为了整个CI/CD的入口，现在因为重启失败，已经激起民怨了。

打开浏览器，就是Jenkins正在重启的首页，等待10分钟左右也不见好转。查看log，也是没有任何错误信息。百思不得其解，拷贝jenkins.war到备份服务器，运行war包。仍然是卡死在jenkins重启的界面，难道今天Jenkins要放假？

本着程序员不信玄学的精神，坚持为人民服务的思想，我把war包scp到本地，运行war包，依然是重启的界面。

多年以后，你如果想问我是怎么解决这个问题的。我可能会说，我关掉了WIFI。

Jenkins日志出现了，连接update center失败的信息，原来重启后，需要访问 update.jenkins-ci.org 。然而在1024这天，这个网站被墙了。

配置 update.jenkins-ci.org 指向一个不存在的IP，快速失败就能进入Jenkins了。

# jenkins
docker网络
docker run命令
